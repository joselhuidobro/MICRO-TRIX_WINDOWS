TRIX: Educación Tecnológica

Curso de Robótica y Controladores DEV: DRONE

Aprende a programar sin Arduino usando ESP-IDF en un entorno profesional.

Domina Docker, Docker Compose, UDP y ROS 2.

Programa un ESP32 con los sensores necesarios.

Duración: 40 horas.

Modela y simula tu dron en Blender.

Interfaz gráfica FLASK.

Curso presencial.

por que es importante este curso

  • Arduino te limita cuando necesitas control de bajo nivel; con ESP-IDF accedes a todo el hardware.
  • Docker reproduce tu entorno robótico al 100 %, incluidas dependencias y estructura de carpetas.
  • Flask es un framework Python ligero ideal para paneles web e interfaces gráficas.
  • ROS 2 orquesta tus nodos y crea flujos de datos en tiempo real entre sensores y actuadores.
